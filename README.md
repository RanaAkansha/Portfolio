# Akansha Rana - Portfolio

A production-ready portfolio built with React, Tailwind CSS, and Framer Motion. This project showcases my experience, projects, and skills as a Full Stack Developer.

## Features
- **Responsive Design**: Optimized for desktop, tablet, and mobile devices.
- **Dark/Light Mode**: Seamless theme switching with system preference detection.
- **Dynamic Animations**: Fluid transitions powered by Framer Motion.
- **Recruiter-Friendly**: Clean, scannable layout with direct links to live projects, source code, and a downloadable resume.

## Tech Stack
- **Frontend**: React.js, Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Build Tool**: Vite

## Folder Structure
- `public/`: Static assets including images and the downloadable `resume.pdf`.
- `src/components/`: Modular React components divided into `layout`, `sections`, and `ui`.
- `src/data/`: Static data files for content like projects.
- `src/hooks/`: Custom React hooks for theme and scroll management.
- `src/index.css`: Global styles and Tailwind configuration.

## Installation & Running Locally

1. Clone the repository
   ```bash
   git clone https://github.com/RanaAkansha/Portfolio.git
   cd Portfolio
   ```
2. Install dependencies
   ```bash
   npm install
   ```
3. Start the development server
   ```bash
   npm run dev
   ```

## Deployment
This project is optimized for deployment on Vercel, Netlify, or similar static hosting platforms. Run `npm run build` to generate the production build.
